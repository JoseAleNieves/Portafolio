Proyectos

🌴 Aplicación Web – Posada “El Oasis de Jeanmar” 💼
Gestión de reservas y operaciones internas

Descripción del proyecto: Desarrollé una aplicación web integral para la gestión operativa de la posada “El Oasis de Jeanmar”, con el objetivo de automatizar las reservas, mejorar el control de disponibilidad de habitaciones y centralizar las operaciones internas del negocio.

💡 Funciones principales:
✅ Autenticación de usuarios con verificación por correo y control de roles (admin, empleados, clientes).
✅ Calendario interactivo de disponibilidad (FullCalendar).
✅ CRUD completo de reservas, gestión de habitaciones (disponible, ocupada, mantenimiento).
✅ Dashboard para personal con métricas y accesos personalizados.
✅ Catálogo de productos con stock en tiempo real.
✅ Sistema de pagos: integración con pasarela PayPal y módulo de pago móvil.
✅ Consumo de API para mostrar la tasa del dólar (BCV).
✅ Reportes manuales descargables por sección (reservas, productos, etc.).
✅ Bitácora de eventos del sistema (acciones de usuarios).
✅ Diseño responsivo con Bootstrap 5.

🛠️ Tecnologías utilizadas: HTML · CSS · JavaScript · CodeIgniter 4 · MySQL · Bootstrap 5 · FullCalendar · SweetAlert · TCPDF · Disqus · XAMPP · PayPal API · InfinityFree
